基本情况

侯素娟，女，汉族，山东成武人，中共党员，现为山东师范大学信息科学与工程学院副教授、博士生导师。2015年12月毕业于重庆大学计算机科学与技术专业， 获工学博士学位。2012年获博士生国家奖学金（No.2012-06674）, 2013获中国留学基金委公派留学联合培养资格，同年10月至2015年9月在悉尼科技大学人工智能中心（QCIS）联合培养。 IEEE会员、中国计算机学会多媒体专委会执行委员、中国图学学会理事会图学大数据专委会委员、山东省人工智能学会常务理事。 在Pattern Recognition、AAAI、IEEE Trans. on Multimedia、ACM MM等国际高水平期刊和会议上发表论文40余篇，是CVPR、IJCNN、IJCAI、Neurocomputing、 Multimedia Tools and Applications、IEEE Multimedia等多个期刊和会议的审稿人和PC Member。 目前主持国家级项目3项、省部级项目2项、厅局级项目1项，参与国家级和省级项目若干项。 申请专利17项（已授权4项），指导全国大学生数学建模大赛获山东赛区二等奖3项，三等奖2项。

资源下载

Logo-2K+:大规模公开的真实世界Logo分类数据集，包含2,341个类别和167,140张图像。与现有的流行的Logo数据集（如FlickrLogos-32和LOGO-Net）相比，Logo-2K+有更全面的标志类别覆盖和更多的标志图像。
论文：《Logo-2K+: A Large-Scale Logo Dataset for Scalable Logo Classification》
数据集下载：https://github.com/housujuan123/Logo-2k-plus-Dataset

LogoDet-3K:目前最大的具有完备标注的Logo检测数据集，含3000个类别、大约20万个手动注释的Logo对象和158,652张图像。LogoDet-3K为Logo检测创建了一个更具有挑战性的基准，因为与现有数据集相比，它在Logo类别和注释对象中具有更高的覆盖率和更广泛的多样性。
论文：《LogoDet-3K: A Large-scale Image Dataset for Logo Detection》
数据集下载：https://github.com/housujuan123/LogoDet-3K-Dataset

FoodLogoDet-1500:第一个最大的公开提供的高质量的食品Logo检测数据集， 含有1500个类别，约10万张图像和15万个手动标注的食品Logo对象。
论文：《FoodLogoDet-1500: A Dataset for Large-Scale Food Logo Detection via Multi-Scale Feature Decoupling Network》
数据集下载：https://github.com/housujuan123/FoodLogoDet-1500-Dataset
